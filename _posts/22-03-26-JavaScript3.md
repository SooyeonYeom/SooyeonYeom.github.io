### 생성자 (Construtor) 란?

  + 정의 : 미리 정해진 Key-Value 값을 지닌 객체를 편리하게 생성할 수 있게 도와주는 객체
  + 쓰임 : 비슷한 객체들을 여러개 생성할 때, 하나의 생성자를 만들어 놓으면 그 생성자를 이용하여 간편하게 여러 인스턴스를 생성할 수 있다.
  + 형식 : 생성자의 첫 시작은 대문자로 시작하는 관례가 있다.
  + 방법 : 
    - 생성자 (Key값1, Key값2, Key값3...) { }
    - 생성자 내부에서 this.로 접근해 각 Key값에 Value값 할당 
    - new 생성자 (속성값1, 속성값2, 속성값3...) 로 객체 생성
